A guy walks into an inherited codebase and orders an integer range. RangeUtils replies:

    import guava.*;

    public List<Integer> getRange(int range) {
        return Ints.asList(Ints.toArray(ContiguousSet.create(Range.closed(0, range - 1), DiscreteDomain.integers())));
    }
